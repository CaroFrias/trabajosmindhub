El elemento input es el mas utilizado, se puede mostrar de muchas formas según el atributo type en este link se puede apreciar https://www.w3schools.com/html/html_form_input_types.asp

El elemento label define una etiqueta para varios elementos del formulario, es útil para usuarios con lectores de pantalla (lee en voz alta la etiqueta) cuando el usuario se posicione en el elemento de entrada.

El atributo for de la etiqueta label debe ser igual al id del input para unirlos

Consideración se debe tener en cuenta que cada input debe tener el atributo name para ser enviado, si éste atributo se omite, el valor del campo de entrada no se enviará.

El atributo type="number" define un campo de entrada de tipo numérico

El atributo type="date" define un campo de entrada de tipo fecha

El tipo checkbox (atributo type="checkbox") le muestra al usuario seleccionar o no una de las opciones determinadas

Elemento select

El elemento select define una lista desplegable, dentro de los elementos select están los elementos option que son las opciones de la lista desplegable y se pueden seleccionar de forma predeterminada, se selecciona el primero elemento de la lista desplegable.


Para definir una opción preseleccionada se debe agregar el atributo selected

El input type="submit" (botón enviar) define el botón para enviar los datos del formulario.
El controlador del formulario suele ser un archivo en el servidor con un script para procesar los datos de entrada.
El manejador de formulario se especifica en el atributo action del formulario.
El atributo target especifica donde mostrar la respuesta _blank (pagina nueva), _self (por defecto en la misma ventana)
